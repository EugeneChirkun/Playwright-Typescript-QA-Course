# Course Structure

| Module | Topic | Main focus | Result |
| --- | --- | --- | --- |
| Module 01 | Environment: Node.js, npm, VS Code и Playwright | Подготовка рабочего окружения | Студент готов установить tools и проверить базовый setup. |
| Module 02 | Git: branches и основные команды | Базовый Git workflow | Студент уверенно работает с branch, commit и pull request. |
| Module 03 | Введение в TypeScript | Синтаксис и типизация | Студент понимает базовые TypeScript concepts. |
| Module 04 | TypeScript для QA | Типы и структуры для test automation | Студент применяет TypeScript в QA scenarios. |
| Module 05 | Async, await и promises | Асинхронность в tests | Студент понимает async flow в automation tests. |
| Module 06 | Основы OOP | Classes и objects | Студент готов к Page Object Model. |
| Module 07 | Основы Playwright | Первые UI tests | Студент пишет базовые Playwright tests. |
| Module 08 | Locators, assertions и auto-waiting | Стабильные проверки UI | Студент выбирает надежные locators и assertions. |
| Module 09 | Page Object Model и clean code | Структура test project | Студент проектирует поддерживаемые page objects. |
| Module 10 | Fixtures, test data и auth | Переиспользование setup и данных | Студент управляет fixtures, auth и test data. |
| Module 11 | API testing with Playwright | API checks внутри Playwright | Студент пишет базовые API tests. |
| Module 12 | CI и final project | GitHub Actions и финальный проект | Студент собирает итоговый mini-project и понимает CI. |
