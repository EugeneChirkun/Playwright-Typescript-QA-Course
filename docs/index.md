# Playwright TypeScript QA Course

## Course goal

Помочь опытному Manual QA engineer перейти в Automation QA через практику с TypeScript и Playwright.

## Target audience

Курс рассчитан на Manual QA engineers, которые уверенно понимают тест-дизайн, баг-репорты и QA-процессы, но только начинают писать automation tests.

## Expected result

Через 3 месяца студент сможет самостоятельно создавать базовый Playwright project, писать UI и API tests, использовать Page Object Model, работать с fixtures, test data, Git, branch, commit и pull request.

## Learning format

- Короткая теория, достаточная для понимания и interview preparation.
- Практика после каждой важной темы.
- Code examples и небольшие QA use cases.
- Homework после каждого module.
- Review по checklist и pull request.

1. Проходите modules по порядку.
2. Читайте overview module и выполняйте practice during the lesson.
3. Делайте homework в отдельном branch.
4. Проверяйте себя по checklist.
5. Используйте resources для official documentation и дополнительных материалов.
