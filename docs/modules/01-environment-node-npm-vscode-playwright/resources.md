# Ресурсы: Модуль 01

## Официальная документация

- [Официальный сайт Node.js](https://nodejs.org/) — официальный сайт Node.js, где можно скачать LTS-версию и проверить актуальную информацию по установке.
- [Документация npm package.json](https://docs.npmjs.com/cli/v10/configuring-npm/package-json) — документация npm о структуре `package.json`, скриптах и зависимостях.
- [Документация npm scripts](https://docs.npmjs.com/cli/v10/using-npm/scripts) — описание npm-скриптов и того, как npm запускает команды из `package.json`.
- [Документация VS Code](https://code.visualstudio.com/docs) — официальная документация VS Code по редактору, терминалу, расширениям и базовой работе с проектом.
- [Стартовое руководство Playwright](https://playwright.dev/docs/intro) — официальное стартовое руководство по установке Playwright и созданию первого проекта.
- [Документация расширения Playwright для VS Code](https://playwright.dev/docs/getting-started-vscode) — инструкция по работе с Playwright через расширение VS Code.
- [Документация Playwright test runner](https://playwright.dev/docs/running-tests) — документация по запуску тестов, выбору файлов тестов и основных команд CLI.
- [Документация Playwright HTML report](https://playwright.dev/docs/test-reporters#html-reporter) — описание HTML reporter и команды для открытия отчёта после запуска тестов.
