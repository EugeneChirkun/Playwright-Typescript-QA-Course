# Ресурсы: Модуль 01

## Node.js и npm

- [Загрузка Node.js](https://nodejs.org/en/download) — официальная страница загрузки Node.js. Для курса выбирайте LTS-версию.
- [Документация npm package.json](https://docs.npmjs.com/cli/v10/configuring-npm/package-json) — документация npm о структуре `package.json`, скриптах и зависимостях.
- [Документация npm scripts](https://docs.npmjs.com/cli/v10/using-npm/scripts) — описание npm-скриптов и того, как npm запускает команды из `package.json`.

## VS Code

- [Загрузка VS Code](https://code.visualstudio.com/download) — официальная страница загрузки редактора.
- [Документация VS Code](https://code.visualstudio.com/docs) — официальная документация VS Code по редактору, терминалу, расширениям и базовой работе с проектом.
- [Встроенный терминал VS Code](https://code.visualstudio.com/docs/terminal/basics) — официальное руководство по открытию и использованию терминала в редакторе.
- [Playwright Test for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-playwright.playwright) — официальное расширение для запуска и отладки тестов Playwright из VS Code.

## Playwright

- [Стартовое руководство Playwright](https://playwright.dev/docs/intro) — официальное руководство по установке Playwright и созданию первого проекта.
- [Видео с установкой Playwright](https://youtu.be/SLhz2KmBh2Q) — короткий пример установки Playwright и первого запуска.
- [Официальный канал Playwright с плейлистами](https://www.youtube.com/@Playwrightdev/playlists) — плейлисты от команды Playwright с обучающими материалами.
- [Документация расширения Playwright для VS Code](https://playwright.dev/docs/getting-started-vscode) — инструкция по работе с Playwright через расширение VS Code.
- [Документация Playwright test runner](https://playwright.dev/docs/running-tests) — документация по запуску тестов, выбору файлов тестов и основных команд CLI.
- [Документация Playwright HTML report](https://playwright.dev/docs/test-reporters#html-reporter) — описание HTML reporter и команды для открытия отчёта после запуска тестов.
