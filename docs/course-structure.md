# Структура курса

| Модуль | Тема | Основной фокус | Результат |
| --- | --- | --- | --- |
| Модуль 01 | Environment: Node.js, npm, VS Code и Playwright | Подготовка рабочего окружения | Студент готов установить инструменты и проверить базовую настройку. |
| Модуль 02 | Git: branches и основные команды | Базовый Git workflow | Студент уверенно работает с branch, commit и pull request. |
| Модуль 03 | Введение в TypeScript | Синтаксис и типизация | Студент понимает базовые концепции TypeScript. |
| Модуль 04 | TypeScript для QA | Типы и структуры для test automation | Студент применяет TypeScript в QA-сценариях. |
| Модуль 05 | Async, await и promises | Асинхронность в тестах | Студент понимает async flow в автотестах. |
| Модуль 06 | Основы OOP | Classes и objects | Студент готов к Page Object Model. |
| Модуль 07 | Основы Playwright | Первые UI-тесты | Студент пишет базовые тесты на Playwright. |
| Модуль 08 | Locators, assertions и auto-waiting | Стабильные проверки UI | Студент выбирает надежные locators и assertions. |
| Модуль 09 | Page Object Model и clean code | Структура тестового проекта | Студент проектирует поддерживаемые Page Object. |
| Модуль 10 | Fixtures, test data и auth | Переиспользование настройки и данных | Студент управляет fixtures, auth и test data. |
| Модуль 11 | API testing with Playwright | API-проверки внутри Playwright | Студент пишет базовые API-тесты. |
| Модуль 12 | CI и финальный проект | GitHub Actions и финальный проект | Студент собирает итоговый мини-проект и понимает CI. |
