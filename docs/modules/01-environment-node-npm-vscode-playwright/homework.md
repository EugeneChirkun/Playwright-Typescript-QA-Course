# Домашнее задание: Модуль 01

## Задания

- Повторить материалы модуля.
- Выполнить небольшое практическое задание по теме `Environment: Node.js, npm, VS Code и Playwright`.
- Зафиксировать вопросы, которые появились во время выполнения.

Студент показывает, что понял цель модуля и может применить тему в простом QA-сценарии.

## Как отправить

Создать отдельную ветку (branch), сделать commit с выполненным заданием и открыть pull request на ревью.

## Чеклист для проверки

- Задание выполнено полностью.
- Решение оформлено понятно.
- Есть краткое описание выполненных шагов.
- Вопросы или блокеры указаны в pull request, если они есть.
