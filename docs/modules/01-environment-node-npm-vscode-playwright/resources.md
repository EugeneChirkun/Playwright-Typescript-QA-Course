# Ресурсы: Модуль 01

## Официальная документация

- [Node.js official website](https://nodejs.org/) — официальный сайт Node.js, где можно скачать LTS version и проверить актуальную информацию по установке.
- [npm package.json docs](https://docs.npmjs.com/cli/v10/configuring-npm/package-json) — документация npm о структуре `package.json`, scripts и dependencies.
- [npm scripts docs](https://docs.npmjs.com/cli/v10/using-npm/scripts) — описание npm scripts и того, как npm запускает команды из `package.json`.
- [VS Code docs](https://code.visualstudio.com/docs) — официальная документация VS Code по editor, terminal, extensions и базовой работе с project.
- [Playwright getting started](https://playwright.dev/docs/intro) — официальный стартовый guide по установке Playwright и созданию первого project.
- [Playwright VS Code extension docs](https://playwright.dev/docs/getting-started-vscode) — инструкция по работе с Playwright через VS Code extension.
- [Playwright test runner docs](https://playwright.dev/docs/running-tests) — документация по запуску тестов, выбору test files и основным CLI commands.
- [Playwright HTML report docs](https://playwright.dev/docs/test-reporters#html-reporter) — описание HTML reporter и команды для открытия report после запуска тестов.
