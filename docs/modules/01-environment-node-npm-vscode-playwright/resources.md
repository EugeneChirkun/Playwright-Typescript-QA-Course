# Ресурсы: Модуль 01

## Официальная документация

- [Node.js official website](https://nodejs.org/) — официальный сайт для установки Node.js; для курса рекомендуется LTS-версия.
- [npm package.json docs](https://docs.npmjs.com/cli/v10/configuring-npm/package-json) — описание структуры `package.json`, dependencies и scripts.
- [npm scripts docs](https://docs.npmjs.com/cli/v10/using-npm/scripts) — официальное объяснение npm scripts и жизненного цикла команд.
- [VS Code docs](https://code.visualstudio.com/docs) — документация по VS Code, терминалу, расширениям, поиску и работе с проектом.
- [Playwright getting started](https://playwright.dev/docs/intro) — официальный старт Playwright: установка, первый проект и запуск тестов.
- [Playwright VS Code extension docs](https://playwright.dev/docs/getting-started-vscode) — как использовать Playwright Test for VS Code extension для запуска и отладки тестов.
- [Playwright test runner docs](https://playwright.dev/docs/test-intro) — базовое описание раннера тестов Playwright, структуры тестов и запуска.
- [Playwright HTML report docs](https://playwright.dev/docs/test-reporters#html-reporter) — как открыть и использовать HTML report после запуска тестов.
